Fuji Electric PIM (Power Integrated Modules)

Power integrated modules combining IGBT and diode/rectifier chips in a single compact package

Fuji Electric's Power Integrated Modules (PIM) are advanced power modules that combine IGBTs and diodes/rectifiers in a single package. This integration provides compact, cost-effective solutions for applications requiring both inverting and rectifying functions, such as in air conditioning systems and industrial drives.

Key Features

  • Integrated inverter and rectifier functions
  • Compact design reducing system size
  • Improved thermal performance
  • Reduced component count and assembly time
  • Enhanced system reliability

Applications

  • Commercial air conditioning systems
  • Industrial motor drives
  • Heat pump systems
  • General purpose inverters

1-Unit IGBT Modules

Single switch modules for custom topologies and specialized applications

Fuji Electric's 1-unit IGBT modules contain a single IGBT and diode in a single package. These modules offer the flexibility of discrete devices with the convenience and improved thermal characteristics of a module package. Ideal for custom topologies and specialized applications.

Key Features

  • Single IGBT with anti-parallel diode
  • Multiple package options
  • High current density options
  • Optimized for custom circuit topologies
  • Available with 7th and 6th Generation chips

Applications

  • Custom converter topologies
  • Resonant circuits
  • Class D amplifiers
  • Welding equipment
  • Induction heating

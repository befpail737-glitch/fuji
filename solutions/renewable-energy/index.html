Inverter System Components

A typical solar inverter system includes:

  • PV input section with MPPT circuits
  • DC boost stage with PFC functionality
  • DC/AC inversion stage with Fuji Electric IGBT modules
  • Isolation transformer (in some topologies)
  • Grid connection circuitry with protection
  • Control board with monitoring and safety functions

Recommended Components

  • 7th Generation X-Series IGBTs (e.g. 2MBI100XAA-50) for main inverter stage
  • Diodes for free-wheeling applications
  • MOSFETs for DC/DC conversion stages
  • Supporting components for protection and filtering

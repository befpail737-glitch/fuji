Drive System Components

Our industrial drive solutions typically include:

  • Rectifier stage for AC-to-DC conversion
  • DC link with filtering components
  • Inverter stage with Fuji Electric IGBT modules
  • Control board with microcontroller and gate drivers
  • Protection and monitoring circuits
  • Human Machine Interface (HMI) for operation

Recommended Components

  • IGBT modules for inverter stage (e.g. 2MBI100XAA-50)
  • IPM for lower power servo applications
  • Discrete MOSFETs for specific circuits
  • Supporting components for protection and filtering
